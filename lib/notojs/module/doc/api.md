## NotoJS APIs

**NotoJS** exposes APIs at several levels depending on how you integrate with it. The HTTP API is used by external tools and clients to communicate with the running server. The JavaScript API is intended for displaying **NotoJS** notebooks output on external web pages. The C++ API is intended for native extensions, server-side plugins, and embedded integrations that need direct access to **NotoJS**, **QuickJS**, and host-backed objects.
